1-  Verifica se houve submissão de dados:

    isset(x) retorna true se x foi iniciado

    empty(x) retorna true se x estiver vazio

    if(isset($_POST) and !empty($_POST)){
        ...sucesso...
    } else{
        ...falhou...
    }

2- Verifica se a submissão foi feita especificamente pelo botão submit correto (name do submit = x e seu value = y )

    if(isset($_POST['x']) <> 'y'){
        ...O botão não foi utilizado...
    }

3- Ao coletar dados dos campos, sempre utilize a função strip_tags para evitar ataques xss

    $exemplo = strip_tags($_POST['exemplo'])

    Assim qualquer script dentro do campo não será executado como uma tag

4- Uma forma similar ao strip_tag:

    $exemplo = htmlspecialchars($_POST['exemplo'], ENT_QUOTES, 'UTF-8');

    Qualquer script dentro do campo não será executado como script e será substituído por respectivos chars
